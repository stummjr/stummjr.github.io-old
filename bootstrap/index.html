Framework CSS - Bootstrap

O bootstrap é um framework CSS para o desenvolvimento de projetos web responsivos.

  • É um conjunto de recursos para serem utilizados em páginas HTML: classes CSS para personalizar o visual e o comportamento de componentes
  • Cria um visual consistente através de elementos visuais para a maioria das tags HTML
  • Ícones, grids, botões, etc.
Este material é uma adaptação da documentação oficial, disponível em: getbootstrap.com/css

Como utilizar o Bootstrap?

Instalação

Baixe o pacote Bootstrap em: getbootstrap.com/getting-started/#download

Utilização

Para usá-lo, extraia as 3 pastas do bootstrap (css, js e fonts) em um diretório acessível a partir do diretório do seu projeto e então inclua os arquivos na sua página:

<link rel="stylesheet" href="css/bootstrap.css" />

No exemplo acima, considere que a pasta css está dentro da mesma pasta do arquivo HTML em questão.

Feito isso, os elementos HTML que você adicionar à sua página passarão a ter o estilo padrão do Bootstrap.

Testando o Bootstrap

Teste o código abaixo em uma página com o bootstrap incluso e também em uma página sem o bootstrap (utilizando as definições-padrão do web browser):

<h1>Hello, world!</h1>
<p>Testando um recurso simples do Bootstrap.</p>

Incluindo o jQuery no seu projeto

Atenção! Para tirar maior proveito do Bootstrap, é preciso incluir no seu HTML o framework JavaScript jQuery. Para tanto, insira no final do seu arquivo HTML (pode ser a última instrução antes de fechar o body) o seguinte conteúdo:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js">

Para obter esse efeito de grid, é preciso criar um div com a classe container, e dentro desse div colocar um outro div com a classe row. Depois disso, é só definir a divisão de colunas, através de divs com classes col-sm-x, onde x é um número. A soma dos x de um row deverá ser 12. Veja o código do grid de 3 partes iguais de 4 colunas: 

<div class="row">
    <div class="col-sm-4">.col-sm-4</div>
    <div class="col-sm-4">.col-sm-4</div>
    <div class="col-sm-4">.col-sm-4</div>
</div>

Em outro exemplo, temos um leiaute dividido em 2 partes diferentes, um ocupando 4 colunas e outro ocupando 8 colunas.

.col-sm-4
.col-sm-8

O código que gerou tal leiaute é: 

<div class="row">
    <div class="col-sm-4">.col-sm-4</div>
    <div class="col-sm-8">.col-sm-4</div>
</div>

Deslocando colunas

As classes col-sm-offset-* permitem que façamos um deslocamento na posição de uma coluna, onde * é a quantidade de colunas que desejamos mover para a direita.

.col-sm-4
.col-sm-4 .col-sm-offset-4

Estados de Validação

Os inputs podem também ter indicações sobre a operação realizada, como: erro, atenção e sucesso. Basta incluir as classes has-error, has-success ou has-warning em um elemento pai, porque daí todos os elementos filhos de classes control-label, form-control e help-block assumirão a cor correspondente ao estado. Ícones

Ícones do pacote glyphicons podem ser adicionados através de elementos span com a classe glyphicon e a classe relativa ao ícone desejado. Por exemplo, o código abaixo gera um botão com um ícone de telefone como rótulo: <button type="button" name="button" class="btn btn-primary"><span class="glyphicon glyphicon-earphone"></span> Call</button>
